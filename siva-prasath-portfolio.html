Predictive Maintenance System Using IoT Sensor Data

Random Forest XGBoost LSTM Seaborn

Built a predictive maintenance system on NASA's CMAPSS Turbofan Engine dataset — processing 20,000+ multivariate sensor readings across 21 channels. Engineered rolling statistics, lag features, and per-operating-mode normalisation before training and benchmarking three models: Random Forest, XGBoost, and an LSTM network. A weighted ensemble achieved 91% precision, predicting failures 10+ cycles in advance and identifying the top five degradation sensor signatures.

Flask API XGBoost Geospatial Python
